Requirements
------------
Python 2.7    
Virtualenv    
Pip    

Installation
------------

Install project by running:

    mkdir QuickFlaskProject
    cd QuickFlaskProject
    git clone git@github.com:juzten/QuickFlask.git
    cd QuickFlask
    virtualenv env
    . env/bin/activate
    pip install -r requirements
    cd ChadevPython
    ./app.py
    
  open web browser to localhost:5000
